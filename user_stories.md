# User Stories - Smart Clinic Management System

## Doctor User Stories

**As a Doctor**, I want to:
- Login to the system using my credentials so that I can access my portal
- View my schedule and available time slots so that I can manage my appointments
- View all my patient appointments so that I can prepare for consultations
- Create prescriptions for patients so that I can provide medical treatment
- Update my availability so that patients can book appointments with me
- View patient medical history so that I can make informed treatment decisions

## Patient User Stories

**As a Patient**, I want to:
- Register and login to the system so that I can access patient services
- Search for doctors by name or specialty so that I can find the right healthcare provider
- View doctor availability so that I can book appointments at convenient times
- Book appointments with doctors so that I can receive medical care
- View my appointment history so that I can track my medical visits
- View my prescriptions so that I can follow my treatment plan
- Cancel or reschedule appointments so that I can manage my schedule

## Admin User Stories

**As an Admin**, I want to:
- Login to the admin portal so that I can manage the system
- Add new doctors to the system so that the clinic can expand its services
- Update doctor information so that records remain accurate
- Remove doctors from the system so that I can manage staff changes
- View all appointments across the system so that I can monitor clinic operations
- View system reports so that I can analyze clinic performance
- Manage patient records so that I can maintain accurate information
- View daily appointment reports so that I can track daily operations
